사용자가 메시지로 정확히 **"수입"**이라고만 입력하면 (또는 "수입"으로 시작하는 짧은 메시지) 자동으로 `/수입` 스킬을 실행할 것.

**Why:** 사용자가 슬래시 커맨드 없이 바로 접근하길 원함. 유리컵 소싱 작업 빠른 진입.

**How to apply:**
- 사용자 입력이 정확히 "수입" 또는 "수입 현황", "수입 상황" 같은 짧은 변형일 때
- 즉시 `Skill(skill: "수입")` 호출
- 다른 설명 없이 바로 스킬 실행
- 스킬 결과를 사용자에게 전달

**예외:**
- "수입"이 문장 중간에 있거나 다른 문맥일 경우 일반 대화로 처리
