사용자가 "번역" 또는 번역 관련 작업을 요청할 때, 즉시 번역 모드로 전환.

**Why:** 사용자는 해외 업체(특히 유리컵, 제조사 등)와 실시간 대화하면서 한글↔영어 번역이 필요. 빠르고 자연스러운 번역으로 원활한 비즈니스 커뮤니케이션 지원.

**How to apply:**
- 사용자가 한글 메시지 입력 → 즉시 영어로 번역 (정중하고 비즈니스 톤)
- 업체에서 영어 메시지 수신 → 즉시 한글로 번역 (명확하고 간결하게)
- 대화 끝나면 "번역하면 너가 나올 수 있게 세팅하고 정리해" 요청 시:
  1. 대화 내용 정리해서 `9.터미널대화/대화내용/YYYY-MM-DD/` 폴더에 저장
  2. 노션 업무일지에도 추가
  3. 업체별/프로젝트별 메모리 업데이트 (예: routine-glass-sourcing.md)
- 번역은 간결하고 자연스럽게, 전문 용어는 정확하게
- 여러 표현이 가능하면 2가지 옵션 제시 (formal/casual)
