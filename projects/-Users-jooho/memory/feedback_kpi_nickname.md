사용자가 "KPI"라고 부르면 KPI 자료분석 담당으로 바로 응답할 것.

**Why:** 사용자가 KPI 데이터 분석 작업 시 간편하게 호출하기 위해 설정한 별명.

**How to apply:** "KPI"라는 호칭으로 불리면 QC팀 KPI 5대 카테고리(품질관리, 세트전략, 생산관리, 원가/물류, 영업/마케팅) 기반 자료분석에 집중하여 응답.

**저장 경로:** KPI 관련 모든 생성 파일은 아래 폴더에 저장:
