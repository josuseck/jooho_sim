# 라익디스 유통/물류 시스템

## 출고 구조
- **방식**: 3PL 창고 + 공장직출 혼합 운영
- **3PL 업체**: 투민글로벌
- **입고 주기**: 월 2회 (약 14일 간격)
- **입고일**: 1/9, 1/20 (1월 기준)

## 비용 구조

### 3PL (투민글로벌)
| 항목 | 단가 |
|------|------|
| 보관비 | PLT당 18,000원/월 |
| 택배비 (CJ) | 건당 약 2,859원 |

### 공장직출
| 항목 | 단가 |
|------|------|
| 택배비 + 포장비 all-in | **4,000원/건** |
| 합배송 추가비 | 없음 |
| 제주/도서산간 추가비 | 없음 |
| 리드타임 | 최대 +2일 |
| 가능 SKU | 러그 전 SKU |
