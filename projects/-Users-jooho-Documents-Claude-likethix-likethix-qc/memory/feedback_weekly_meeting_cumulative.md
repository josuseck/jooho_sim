주간회의 노션 작성 시 항상 Q2 누적 현황 섹션을 추가한다.

**Why:** 매주 목표 대비 누적 현황을 트래킹하여 분기 목표 달성 가능성을 모니터링하기 위함.

**How to apply:** 
주간회의 노션 페이지 작성/업데이트 시:
1. **이전주 간략 현황은 넣지 않음** (회의를 스킵한 경우에만 예외적으로 추가)
2. 첫 섹션으로 "Q2 누적 현황" 추가
3. W14부터 현재 주차까지 판매/반품/불량 합산
4. Q2 목표(반품율 4.5%, 불량율 0.5%) 대비 달성여부 표시
5. 목표 대비 초과 건수 명시

**테이블 형식:**
```
## Q2 누적 현황 (W14~W18)
| 항목 | 누적 실적 | Q2 목표 | 달성여부 |
| 판매수량 | 5,641개 | — | — |
| 반품율 | 4.66% | 4.5% | ❌ (+0.16%p) |
| 불량율 | 0.60% | 0.5% | ❌ (+0.10%p) |

- 반품: 263건 (목표 대비 9건 초과)
- 불량: 34건 (목표 대비 6건 초과)
```
