# 자료분석 가이드 참조

사용자가 "자료분석", "데이터 분석", "주간 분석", "판매 분석" 등을 요청하면 자동으로 다음 파일을 읽어서 작업한다:

**위치**: `docs/자료분석_가이드.md`

이 가이드에는 다음 정보가 포함됨:
- 📊 Supabase 데이터베이스 정보
- 🏷️ 바코드 패턴 (LE10400=그레인, LE10300=시어서커, 순서 중요!)
- 📅 주차 기준 (목~수, 회의 주차 = 데이터 +1)
- 📈 분석 항목 (판매/반품/불량 계산식)
- 🎯 Q2 KPI 목표 (반품 4.5%, 불량 0.5%)
- 📝 주간회의 노션 작성 형식
- 🔧 분석 스크립트 경로
- 📂 결과 저장 경로

## 추가 참조 문서

- `docs/데이터베이스_조회_가이드.md` - SQL 쿼리 상세
- `config/barcode_patterns.json` - 바코드 패턴 JSON
- `scripts/analysis/README.md` - 스크립트 사용법
