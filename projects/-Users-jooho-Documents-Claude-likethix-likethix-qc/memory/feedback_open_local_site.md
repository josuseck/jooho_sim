사용자가 "문 열어", "사이트 열어" 등의 요청을 하면 자동으로 로컬 개발 서버를 확인하고 시작

**Why:** 매번 개발 서버 상태를 확인하고 수동으로 시작하는 반복 작업 제거

**How to apply:**
1. 사용자가 "문 열어", "사이트 열어", "페이지 열어" 등의 요청을 하면
2. 먼저 `curl -s http://localhost:3000` 또는 `lsof -i :3000`으로 개발 서버 실행 여부 확인
3. 실행 중이 아니면 `npm run dev` (백그라운드)로 개발 서버 시작
4. 서버가 준비되면 `open http://localhost:3000`으로 브라우저 열기
5. 이미 실행 중이면 바로 브라우저만 열기
