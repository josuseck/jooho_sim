방석/쿠션 충전재가 해외 수입에서 국내 제품으로 전환되었다. 6/10까지는 해외 충전재, 6/11부터 국내 충전재 사용.

**충전재 바코드 매핑:**

| 구분 | 해외 수입 | 국내 | 비고 |
|------|----------|------|------|
| 방석 600g | LE101C2000001 | LE101K1000001 | 6/10까지 해외 |
| 펫쿠션 Medium 650g | LE201C1000002 | LE201K1000002 | 6/11부터 국내 |
| 펫쿠션 Large 1100g | LE201C1000001 | LE201K1000001 | |

**Why:** 
- 6/10: 작업 완료 (어제)
- 6/11: 출고부터 국내 충전재 적용 (오늘부터)
- 원가, 품질, 공급 안정성 등의 이유로 국내 전환

**How to apply:**
- 6/10 이전 데이터: 해외 충전재 바코드 (LE101C2000001 등) 정상
- 6/11 이후 데이터: 국내 충전재 바코드 (LE101K1000001 등) 사용
- 세트 구성품도 함께 변경 예상 (새 세트 바코드 사용)
- 구 세트 바코드(MCSB1004001 등)는 6/9~10 데이터에만 존재
- 관련 메모리: [[project_old_cushion_set_barcodes]]
