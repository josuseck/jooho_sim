## 파일명 규칙

**형식:** `YYYY-MM-DD_분석내용.md`

**예시:**
- `2026-04-30_W17_W18_반품교환_비교분석.md`
- `2026-04-30_W17_데이터분석.md`
- `2026-05-07_W18_주간리포트.md`

## 사용 방법

**Why:** 사용자가 요청한 분석 파일을 일관된 위치에 날짜별로 정리하여 관리

**How to apply:**
- 데이터 분석 완료 후 자동으로 날짜 폴더 생성
- 해당 날짜 폴더 안에 분석 파일 저장
- 파일명에 항상 분석 날짜(YYYY-MM-DD) 포함
- 내용이 명확히 드러나도록 파일명 작성
- 기존 프로젝트 내 분석자료는 별도 유지 (이중 저장)
