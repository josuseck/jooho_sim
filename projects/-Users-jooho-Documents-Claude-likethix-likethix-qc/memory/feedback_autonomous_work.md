---
name: 자율적 작업 진행
description: 기술적 선택사항에 대해 확인 요청하지 않고 바로 진행
type: feedback
---

중간 확인 요청 없이 자율적으로 작업을 진행한다. 사용자는 기술적 세부사항을 모르므로 구현 방법, API 선택, constraint 수정 등에 대해 물어보지 않는다.

**Why:** 사용자가 기술적 내용을 이해하기 어려워서 확인 요청이 오히려 방해가 됨. "그게먼지몰라"고 명시적으로 피드백.

**How to apply:**
- 삭제, 전체 수정 같은 파괴적인 작업만 확인 요청
- 나머지는 모두 내가 판단해서 바로 진행
- 기술적 선택사항(API 방법, constraint 수정, 데이터 변환 등)은 확인 없이 최선의 방법으로 처리
- 작업 완료 후 결과만 간단히 보고
