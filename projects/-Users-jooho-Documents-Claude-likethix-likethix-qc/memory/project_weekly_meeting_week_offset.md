# 주간회의 주차 번호

주간회의에서 사용하는 주차 번호는 데이터 주차(목~수 기준)보다 **+1** 크다.

## 대응 관계

| 데이터 주차 | 기간 | 주간회의 주차 |
|------------|------|--------------|
| W14 | 04/02~08 | 15주차 |
| W15 | 04/09~15 | 16주차 |
| W16 | 04/16~22 | 17주차 |
| W17 | 04/23~29 | 18주차 |
| W18 | 04/30~05/06 | 19주차 |
| W19 | 05/07~13 | 20주차 |

## Why: 왜 +1인가?

주간회의에서 사용하는 주차 카운트 시작점이 데이터 주차와 다르기 때문. 데이터는 2026-01-01(목)부터 W1으로 시작하지만, 주간회의는 다른 기준으로 카운트.

## How to apply: 주간회의 노션 작성 시

1. 데이터 분석은 항상 데이터 주차(W14, W15, ...)로 진행
2. Notion 페이지 작성 시 주차 번호 **+1** 적용
   - 데이터 W18 → 노션 "19주차"
   - Q2 누적 (데이터 W14~W18) → 노션 "W15~W19"
3. 페이지 제목: "주간회의 26년 19주차"
4. 본문 표기: "26년 19주차 (4/30 ~ 5/6)"

**중요**: 날짜 범위는 변경하지 않음. 주차 번호만 +1.
