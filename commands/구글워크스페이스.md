# 구글 워크스페이스 컨텍스트 호출

라익디스 구글 워크스페이스 관련 작업 시 사용. 다음 메모리 파일을 즉시 참조해서 답변할 것:

- `~/.claude/projects/-Users-jooho/memory/google-workspace-setup.md`

## 답변 시 포함할 내용

먼저 현재 상태 요약:
- ✅ likethix.co.kr → 구글 워크스페이스 (Gmail) 셋업 완료
- ✅ likethix.com → 네이버웍스 (그대로 유지)
- ✅ 사용자: `jooho@likethix.co.kr`, `jookyu@likethix.co.kr` (둘 다 Super Admin)
- ⏳ D-Day: 향후 도메인 swap 예정 (com ↔ co.kr)

그 다음 사용자의 구체적 요청에 답변:
- DNS 변경: 카페24 또는 네이버웍스 어디서? 위험성?
- 메일 설정: MX, SPF, DKIM 등
- 사용자/권한: 어드민 콘솔
- D-Day 진행: 단계별 가이드

## 주의사항 (반드시 확인)

⚠️ **likethix.com은 절대 만지지 말 것** - 메일+웹사이트 동시 운영 중, 외부 네임서버 사용
⚠️ likethix.co.kr 네임서버 변경 금지 - 구글 메일 끊김
⚠️ DNS 작업 후 항상 `dig`로 실제 반영 확인

## 자주 쓰는 명령어

```bash
dig NS likethix.co.kr +short
dig MX likethix.co.kr +short
dig TXT likethix.co.kr +short
dig MX likethix.com +short
```

사용자 요청을 확인하고 메모리 참조 후 구체적으로 안내해줘.
