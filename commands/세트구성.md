## 🛠️ 표준 분석 스크립트 템플릿

```python
import pandas as pd
import re

# 1. 판매 데이터 로드
files = [...]  # 분석 기간의 월별 파일
df = pd.concat([pd.read_excel(f) for f in files], ignore_index=True)

# 2. 시어서커 필터링
seersucker = df[df['상품명'].str.contains('시어서커', na=False)]

# 3. 사이즈 추출
def extract_size(option):
    match = re.search(r'(\d+)\s*[xX×]\s*(\d+)', str(option))
    return f"{int(match.group(1))}x{int(match.group(2))}" if match else None

# 4. 주문번호 단위 함께구매 분석
order_groups = seersucker.groupby('주문번호').agg(
    카테고리목록=('카테고리', lambda x: list(x.unique())),
    사이즈목록=('사이즈', lambda x: list(x.dropna().unique())),
    상품수=('카테고리', 'count')
).reset_index()
```

## 🎯 분석 시 주의사항

1. **단순 판매량 ≠ 세트 가능성**
   - 많이 팔리는 사이즈와 함께구매되는 사이즈는 다름
   - 항상 함께구매 데이터로 검증

2. **단독 구매 성향 확인**
   - 80% 이상 단독 구매 카테고리는 세트 부적합
   - 매트리스, 돌소파 등이 해당

3. **시즌 고려**
   - 시어서커: 6-7월 73.6% 집중 (한철 장사)
   - 5월 중순 출시가 골든타임

4. **마진율 정책**
   - 세트 마진율 32~33% 균일 유지
   - 단품 합 대비 약 33% 할인
