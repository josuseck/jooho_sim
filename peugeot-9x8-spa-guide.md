## 푸조 9X8 특징

### 장점
- ✅ **직선 속도 최강** (리어 윙 없음 → 낮은 드래그)
- ✅ 케멜 스트레이트에서 토요타/페라리 따돌림
- ✅ 슬립스트림 효과 극대화

### 단점
- ⚠️ **고속 코너 불안정** (다운포스가 언더바디만 의존)
- ⚠️ **브레이킹 구간 다운포스 부족** (리어 윙 X)
- ⚠️ 급격한 방향 전환 시 언더스티어 발생

## 인게임 세팅 (Fixed Setup 기준)

### 브레이크 바이어스 (BB)

| 상황 | BB | 설명 |
|------|-----|------|
| **베이스** | **52-53%** | 미드십 레이아웃 기본값 |
| **고속 코너 진입** (Pouhon, Blanchimont) | **+1% (53-54%)** | 다운포스 부족 보완 |
| **레이스 후반** (15랩+) | **+1%** | 타이어 마모 시 프론트 그립 부족 |
| **젖은 노면** | **+2-3% (54-56%)** | 리어 휠락 방지 |

### TC (Traction Control) 세팅

**주호 개인 세팅** (공격적 출구 가속):

| 항목 | 설정 | 효과 |
|------|------|------|
| **TC Slip** | **최대** | 휠스핀 많이 허용 → 코너 출구 가속 빠름 |
| **TC Cut** | **기본 -50%** | 부드럽게 개입 → 급격한 파워 컷 방지 |
| **TC Level** | **한 단계 낮게** | 덜 간섭 → 드라이버 컨트롤 증가 |

**세팅 철학**:
- 푸조는 직선 속도가 강점 → TC를 느슨하게 해서 출구 가속 극대화
- TC Cut 50% 줄임 → 파워 전달이 부드러워져서 타이어 마모 감소
- 단점: 스로틀 컨트롤 실수 시 휠스핀 → 타이어 온도 급상승

**일반 세팅** (안정적):
- TC Slip: 중간
- TC Cut: 기본값
- TC Level: 기본값 or +1

## 스파 코너별 상세 공략

### S1 — 라 소스 ~ 오루즈 ~ 케멜

#### 1️⃣ T1 La Source (라 소스 헤어핀)

**특징**: 헤어핀, 60-70km/h, 레이스 최저속 코너

**푸조 공략**:
- **브레이킹 포인트**: 100m 보드 기준
- **브레이킹**: 다른 차보다 **5m 일찍** 시작
  - 이유: 다운포스 부족 → 브레이크 그립 약함
  - MGU 회생 고려 → 압력 85-90%
- **라인**: 바깥 → 안쪽 늦게 터닝 → 넓게 출구
  - 진입: 오른쪽 끝
  - 에이펙스: 왼쪽 안쪽 연석
  - 출구: 오른쪽 넓게 (케멜 진입)

**BB/TC 세팅**:
- **BB**: 51-52% (회전력 필요)
- **TC**: 출구 일찍 스로틀 (케멜 진입 속도 확보)
- **TC Slip 최대**: 출구 휠스핀 허용 → 빠른 가속

**주의사항**:
- **스타트**: 사고 다발 구간 (안전 우선)
- **트레일 브레이킹**: 짧게 (리어 불안정 주의)
- **출구 속도**: 케멜 진입 속도 결정 (중요!)

**노을 시간대 팁**:
- 황금빛 → 서쪽 하늘 밝음 → 가시성 좋음
- 안쪽 연석 그림자 보고 라인

**실전 팁**:
- 출구 속도 1km/h = 케멜 끝 2km/h 차이
- 무리하지 마 (안전 완주 우선)

#### 2️⃣ Eau Rouge / Raidillon (오루즈 / 레디용)

**특징**: 고속 업힐 좌-우, 270-290km/h, 스파 상징

**푸조 공략**:
- **진입 속도**: 270-280km/h (케멜 진입 따라)
- **Eau Rouge** (하단 좌회전):
  - 왼쪽 연석 스쳐 지나감
  - 풀스로틀 유지
- **Raidillon** (상단 우회전):
  - 중앙 → 오른쪽으로 이동
  - 오른쪽 끝 연석까지 사용
  - **푸조는 플랫아웃 가능** ✅

**라인**:
```
왼쪽 (진입) → 중앙 (Eau Rouge) → 오른쪽 (Raidillon 출구)
```

**주의사항**:
- **퀄리**: 신품 타이어 → 플랫아웃 OK
- **레이스 5랩+**: 타이어 마모 → 약간 리프트
- **레이스 후반**: 무조건 리프트 (스핀 = 끝)
- **다운포스 부족**: 차 불안정 → 정확한 라인 필수

**TC**:
- 거의 개입 안 함 (풀스로틀)
- 업힐이라 휠스핀 위험 낮음

**노을 시간대 팁**:
- 상단에서 붉은 노을 보임
- 역광 주의 (서쪽 하늘)
- 라인 집중

**실전 팁**:
- **겁먹지 마**: 푸조는 플랫아웃 가능
- **정확한 라인**: 좌 → 중 → 우 (정확히)
- **실수 시**: 리프트 (무리하지 마)

#### 3️⃣ 케멜 스트레이트 (Kemmel Straight) 🚀

**특징**: 스파 최장 직선, 1.8km, 300km/h+

**푸조 천국** ⭐⭐⭐:
- **직선 속도**: 300-310km/h (최고)
- **토요타 대비**: +8-10km/h 이득
- **페라리 대비**: +3-5km/h 이득
- **시간 절약**: 0.3-0.5초

**공략**:
- **Raidillon 출구 속도**: 케멜 끝 속도 결정
- **슬립스트림**: 앞차 따라붙기 → 추월 각
- **DRS**: 없지만 리어 윙 X → 효과 유사
- **기어비**: 최고단 길게 (리미터 방지)

**오버테이크**:
- **타이밍**: 직선 중반~끝
- **슬립스트림**: 50m 이내 따라붙기
- **Les Combes 진입**: 늦은 브레이킹 시도

**노을 시간대 팁**:
- 직선 → 서쪽 하늘 (붉은 노을)
- 앞차 테일라이트 + 노을 = 분위기 최고

**실전 팁**:
- 여기서 이기면 레이스 이김
- 슬립스트림 = 생명

### S2 — 레 콤 ~ 푸옹

#### 4️⃣ Les Combes (레 콤)

**특징**: 복합 코너 (우-좌), 내리막 브레이킹, 130-180km/h

**푸조 공략**:
- **브레이킹 포인트**: 100m 보드 (300km/h → 130km/h)
- **브레이킹 거리**: 약 150m (엄청 김)
- **내리막 + 다운포스 부족** = 휠락 쉬움

**첫 코너** (우회전, T7):
- **진입 속도**: 130km/h
- **브레이킹**: **일찍 시작** (다른 차 +10m)
- **BB**: 53% (고속 브레이킹 안정성)
- **라인**: 왼쪽 → 안쪽 → 중앙
- **주의**: 프론트 휠락 주의 (끼익 소리)

**두 번째 코너** (좌회전, T8):
- **진입 속도**: 160km/h
- **라인**: 오른쪽 → 안쪽 → 넓게 왼쪽
- **TC**: 출구 일찍 가속 (업힐)
- **주의**: 업힐 휠스핀 위험 ⚠️

**TC 활용**:
- 출구 스로틀 조심 (업힐 + TC Slip 최대 = 휠스핀)
- 부드럽게 가속 (TC Cut -50% 활용)

**노을 시간대 팁**:
- 내리막 → 하늘 보임 (황금빛/붉은빛)
- 브레이킹 포인트 집중 (풍경 구경 X)

**실전 팁**:
- 브레이킹 늦으면 탈출로 → 3초 로스
- 두 번째 코너 출구 = Pouhon 진입 속도

#### 6️⃣ Pouhon (푸옹) ⚠️⚠️⚠️

**특징**: 고속 더블 에이펙스 좌회전, 240-250km/h

**푸조 최대 약점 구간** 💀:
- 고속 + 다운포스 부족 = 언더스티어 심함
- WEC 프로도 조심하는 코너

**진입 속도**:
- **토요타/페라리**: 250km/h
- **푸조**: **240-245km/h** (5-10km/h 느리게!)
- **레이스 후반**: 235-240km/h (타이어 마모)

**브레이킹**:
- 약간 리프트 (완전 브레이크 X)
- BB 53-54% (안정성 우선)

**라인**:
- **바깥 → 첫 에이펙스 → 바깥 → 두 번째 에이펙스 → 바깥**
- 바깥쪽 여유 2m 두기 (언더스티어 대비)
- 안쪽 너무 꽉 붙이면 스핀

**TC 활용**:
- 출구 스로틀 **부드럽게**
- 언더스티어 중 휠스핀 = 스핀 💀
- TC Cut -50% 덕분에 부드러운 파워 전달

**주의사항**:
- **절대 무리하지 마**
- 0.2초 벌려고 무리 → 스핀 → 10초 손해
- 겸손하게 통과 → 다음 코너에서 만회

**노을 시간대 팁**:
- 코너 중간에서 노을 보임 (좌측)
- 풍경 구경 금지 (집중!)

**실전 팁**:
- "Pouhon은 겸손하게" (명언)
- 5km/h 느려도 OK (안전이 우선)
- 출구 속도 > 진입 속도

- **TC**: 스타트 라인에서 휠스핀 주의

## 실전 팁

### 주행
1. **브레이킹은 일찍** (다운포스 부족)
2. **고속 코너는 여유 있게** (언더스티어 예방)
3. **직선에서 싸워라** (푸조 강점)
4. **Pouhon은 겸손하게** (무리하면 스핀)
5. **슬립스트림 = 생명** (추월 기회)

### TC 컨트롤
1. **출구 가속 일찍** (TC Slip 최대 활용)
2. **스로틀 부드럽게** (TC Cut 50% → 급격한 파워 컷 없음)
3. **휠스핀 소리 들으면** 약간 리프트 (타이어 보호)
4. **업힐 구간** 휠스핀 주의 (Les Combes 출구)
5. **젖은 노면** TC Level +2~3 (안정성 우선)

## 브레이크 바이어스 (BB) 완전 가이드

### 기본 개념

**BB %** = 프론트 브레이크에 가는 압력 비율

```
BB 52% = 프론트 52% : 리어 48%
BB 높임 (53%, 54%...) = 프론트 ⬆️, 리어 ⬇️
BB 낮춤 (51%, 50%...) = 프론트 ⬇️, 리어 ⬆️
```

---

### 조절해야 하는 상황 (증상별)

#### 1️⃣ 프론트 휠락 (앞바퀴 잠김)

**증상**:
- 브레이킹 때 **끼익** 소리 (프론트 타이어)
- 차가 직진만 하고 안 돌아감 (언더스티어)
- 코너 진입 놓침

**해결**: **BB -1%** (리어로 이동)

---

#### 2️⃣ 리어 휠락 (뒷바퀴 미끄러짐)

**증상**:
- 브레이킹 때 **뒤가 흔들림**
- 차가 회전하려 함 (오버스티어)
- 스핀 위험 느껴짐

**해결**: **BB +1%** (프론트로 이동)

---

#### 3️⃣ 브레이킹 중 언더스티어

**증상**:
- 브레이크 밟으면서 턴인 시도 → **차가 안 돌아감**
- 트레일 브레이킹 안 먹힘
- 헤어핀 진입 넓어짐

**해결**: **BB -1%** (리어 ⬆️ → 회전력 증가)

---

#### 4️⃣ 브레이킹 중 오버스티어

**증상**:
- 브레이크 밟으면서 턴인 → **뒤가 미끄러짐**
- 스핀할 것 같은 느낌
- 리어 불안정

**해결**: **BB +1%** (프론트 ⬆️ → 안정성 증가)

---

#### 5️⃣ 고속 코너 진입 불안정

**증상**:
- Pouhon, Blanchimont 진입 시 **불안정**
- 차가 흔들림
- 리어가 가벼운 느낌

**해결**: **BB +1%** (안정성 우선)

---

#### 6️⃣ 타이어 마모 (레이스 후반)

**증상**:
- 15랩+ 이후 **프론트 그립 없음**
- 브레이킹 거리 길어짐
- 코너 진입 언더스티어

**해결**: **BB +1%** (리어에 부담 분산)

---

### BB 조절에 따른 차량 거동 변화

#### BB 높임 (프론트 ⬆️) — 예: 52% → 53%

**효과**:
- ✅ **브레이킹 안정성 증가** (직진성 좋음)
- ✅ 리어 휠락 방지 (스핀 위험 감소)
- ✅ 고속 코너 진입 안정적
- ⚠️ **턴인 어려움** (언더스티어)
- ⚠️ 프론트 타이어 부담 증가
- ⚠️ 트레일 브레이킹 효과 감소

**언제 사용**:
- 리어가 불안정할 때
- 고속 코너 진입 (Pouhon, Blanchimont)
- 젖은 노면
- 레이스 후반 (타이어 마모)

---

#### BB 낮춤 (리어 ⬆️) — 예: 52% → 51%

**효과**:
- ✅ **회전력 증가** (차가 잘 돌아감)
- ✅ 트레일 브레이킹 효과 증가
- ✅ 헤어핀 진입 쉬움
- ✅ 프론트 타이어 부담 감소
- ⚠️ **리어 불안정** (스핀 위험)
- ⚠️ 브레이킹 중 뒤 흔들림
- ⚠️ 고속 코너 진입 무서움

**언제 사용**:
- 브레이킹 중 언더스티어
- 헤어핀 많은 트랙
- 프론트 타이어 과열
- 트레일 브레이킹 적극 활용

---

### 왜 1%씩만 조절하는가?

#### 1%가 실제로는 큰 변화

```
BB 52% → 53% (1% 증가)

실제 변화:
프론트: 52% → 53% (+1%)
리어: 48% → 47% (-1%)

→ 프론트는 1% 늘고, 리어는 1% 줄어듬
→ 상대적 차이는 2%포인트!
```

**체감**: 브레이킹 때 확실히 느껴짐

---

#### 3-5% 한 번에 바꾸면 생기는 문제

**예: 52% → 55% (3% 증가)**

**문제**:
- ❌ 프론트 타이어 **휠락 심함**
- ❌ 리어 브레이크가 **거의 안 일함**
- ❌ **새로운 문제 발생** (한 문제 → 두 문제)
- ❌ 브레이킹 파워 **전체적으로 감소**
- ❌ **밸런스 붕괴** → 운전 불가능

**예: 52% → 49% (3% 감소)**

**문제**:
- ❌ 리어 타이어 **휠락 심함**
- ❌ 브레이킹 때 **뒤가 계속 흔들림**
- ❌ **스핀 위험 엄청 높음**
- ❌ 차가 너무 불안정
- ❌ **레이스 망침**

---

#### 밸런스 = 시소 게임

```
     프론트 ←─── 중심 ───→ 리어
        52%              48%

1% 조절:
     프론트 ←─── 약간 ───→ 리어
        53%              47%
        (살짝 기울음, 조절 가능)

3% 조절:
     프론트 ←─── 많이! ───→ 리어
        55%              45%
        (완전 기울음, 밸런스 붕괴)
```

---

#### 1%씩 조절하는 이유 (정리)

1. ✅ **변화의 방향 파악 가능**
   - 좋아졌으면 같은 방향 +1%
   - 나빠졌으면 원래대로 복귀

2. ✅ **되돌리기 쉬움**
   - 1% 조절: 1초
   - 3% 조절: 적정값 찾는 데 10분+

3. ✅ **타이어 보호**
   - 1% 조절: 밸런스 OK
   - 3% 조절: 불균형 마모

4. ✅ **안전**
   - 1% 조절: 적응 가능
   - 3% 조절: 스핀 위험

---

### 실전 조절 방법

#### 1단계: 증상 파악
```
"브레이킹 때 뭐가 불편해?"
- 프론트 휠락? (끼익 소리)
- 리어 불안정? (뒤 흔들림)
- 언더스티어? (안 돌아감)
- 오버스티어? (스핀 위험)
```

#### 2단계: 조절 방향 결정
```
프론트 문제 → BB 낮춰 (리어 ⬆️)
리어 문제 → BB 높여 (프론트 ⬆️)
```

#### 3단계: 한 번에 1%씩
```
52% → 53% (천천히)
절대 52% → 55% 금지! (급격한 변화)
```

#### 4단계: 2-3랩 테스트
```
조절 후 바로 판단 금지
2-3랩 돌면서 체감
```

#### 5단계: 재조정
```
더 나아졌으면: 같은 방향 +1%
악화되었으면: 원래대로 복귀
```

---

### 트레일 브레이킹 관계

**BB 낮음 (리어 ⬆️) = 트레일 쉬움**
- 브레이킹 중 차가 잘 회전
- 리어가 미끄러지며 턴인 도움
- **위험**: 과하면 스핀

**BB 높음 (프론트 ⬆️) = 트레일 어려움**
- 브레이킹 중 차가 직진 경향
- 안정적이지만 회전력 부족
- **안전**: 초보자 추천

---

### 주의 사항

#### ❌ 하지 말아야 할 것
1. **한 번에 3% 이상 조절** (밸런스 붕괴)
2. **매 랩마다 조절** (일관성 없음)
3. **스핀 후 바로 조절** (감정적 판단)
4. **극단적 BB** (48% 이하, 56% 이상)

#### ✅ 해야 할 것
1. **1%씩 조절** (천천히)
2. **2-3랩 테스트** (체감 시간)
3. **증상 파악 먼저** (원인 분석)
4. **메모하기** (어느 BB가 좋았는지)
