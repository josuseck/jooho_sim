## ATSR Hub Evo란?

**ATSR Hub Evo**는 심레이싱 스티어링 휠과 DDU(Digital Display Unit)의 LED를 커스터마이징할 수 있는 범용 애플리케이션입니다.

### 주요 특징
- ✅ **모든 장치 지원**: 현재 출시된 모든 휠/DDU (+ 출시 예정 제품도)
- ✅ **커뮤니티 기반**: 사용자가 만든 프로필 공유
- ✅ **동적 프로필**: 게임별, 차량별 커스터마이징
- ✅ **Lovely Sim Racing 공식 파트너**

---

### 공유 가치

### 통합 멤버십
**모든 Gold Lovely 멤버는 전체 ATSR Hub Evo 플러그인에 무료로 접근 가능**
- Gold 멤버십 = ATSR Hub Evo 전체 기능
- 별도 구매 불필요

## 커뮤니티 프로필

### 프로필 저장소
ATSR Hub는 커뮤니티 기반으로 운영됩니다:
- 🗾 **Track Data**: 트랙별 데이터
- 🚘 **Car Data**: 차량별 데이터
- 💡 **LED Profiles**: LED 프로필

### 프로필 생성/공유
1. **직접 제작**: 동적 프로필 생성 도구
2. **커뮤니티 다운로드**: 미리 만들어진 프로필 사용
3. **즉시 사용 가능**: 장치 출시 당일부터 프로필 사용

## VI-RSR 연동

### VI-RSR LED 구성
- **86개 LED**: 16개 RPM + 70개 기능 LED
- **ATSR Hub Evo**: VI-RSR 전용 프로필 지원
- **SimHub 대비**: 더 많은 커스터마이징 옵션

### 설정 방법
1. ATSR Hub Evo 설치
2. VI-RSR 연결
3. 프로필 선택/제작
4. 게임별 세팅 적용

## 주요 기능

### LED 커스터마이징
- **RPM 바**: 색깔, 밝기, 패턴
- **시프트 알림**: 타이밍, 효과
- **플래그**: 블루/옐로우/레드 플래그 표시
- **기어**: 기어별 색깔 변경
- **DRS/ERS**: 시스템 상태 표시
- **타이어**: 온도/마모 색깔 표시

### 게임별 지원
- **Le Mans Ultimate** (LMU)
- **Assetto Corsa Competizione** (ACC)
- **iRacing**
- **rFactor 2**
- **Automobilista 2**
- **F1 시리즈**

### 동적 프로필
- 차량별 설정
- 트랙별 설정
- 세션별 설정 (연습/퀄리/레이스)

## 트러블슈팅

### 문제: 장치 인식 안 됨
**해결**:
1. USB 재연결
2. ATSR Hub 재시작
3. 장치 드라이버 업데이트

### 문제: LED 안 켜짐
**해결**:
1. 프로필 선택 확인
2. 게임 선택 확인
3. 장치 펌웨어 업데이트

### 문제: 프로필 동작 안 함
**해결**:
1. 게임별 플러그인 활성화
2. 프로필 재다운로드
3. 수동 설정 확인
