## LMU 공방 서버 세팅 팁

- **일몰 레이스**: 일몰 1~2시간 전 시작 → 황혼 구간 포함
- **일출 레이스**: 일출 직전 시작 → 새벽 → 일출 연출
- **르망 24h 스타일**: 15:00 시작 (실제 르망 스타트 시간), 르망 기준 일몰 21:56 / 일출 06:00
- **후지 야간 포함**: 불일치 주의, 실제보다 일몰 빠를 수 있음
- **바레인/루사일**: 일몰이 17~17시대로 빠름 → 야간 레이스 연출 용이
