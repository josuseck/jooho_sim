## SimuCUBE 2 Pro 설정

### Porsche 911 GT3 R 프로파일

```
Overall Strength: 50% (= 12.5Nm)
Reconstruction Filter: 3
Damping: 4%
Friction: 0%
Inertia: 0%
Slew Rate Limit: Disabled
Wheel Range: 540도
Center Spring: 0%
```

### 설치 순서
1. USB 연결 (PC 뒷면 직접)
2. Tuner 3.0 실행 → 펌웨어 업데이트
3. E-Stop 연결 확인
4. High Torque Mode 활성화 (10초 카운트다운)
5. 센터 캘리브레이션 → "Set permanent wheel center"
6. FFB 프로파일 생성

### LMU 진동 이슈 해결
1. **True Drive → Ultra Low Latency OFF** (가장 간단)
2. **lmuffb 앱** (https://github.com/coasting-nc/LMUFFB)
3. 인게임 FFB Smoothing 5~7 (최후 수단)

## 트랙 공략

### 후지 클래식 (LMGT3, Porsche 911 GT3 R)

| 단계 | 랩타임 |
|------|--------|
| 적응 | 1:42대 |
| 안정 | 1:40~1:41대 |
| **현재 베스트** | **1:39.317** |
| 실전 결과 | 1:43.949 (P9, SR +19.3) |
| 목표 (폴포지션) | **1:38대** |
| WEC #92 실차 | 1:40.414 |

**단축 포인트**:
- 헤어핀 (T6-7): 트레일 브레이크 깊게 (0.2~0.3초)
- 새 우측 스위퍼: 샤프 턴인 (0.2초)
- T16 Panasonic: 출구 풀스로틀 0.1초 빠르게 (0.2초)

### Le Mans (Circuit de la Sarthe)

- **길이**: 13.626km, 38 코너
- **LMGT3 랩타임**: 3:50~4:10
- **실전 결과**: 3:47.232 (뮬산, P6, SR +12.6)
- **강점**: Porsche Curves (0.3~0.5초 이득)
- **약점**: Mulsanne 6km 직선 → 톱스피드 부족

### 바레인

- **길이**: 5.412km, 15 코너, 시계 방향
- **성격**: 스톱 앤 고 (브레이킹 핵심)
- **목표 랩타임**: 2:05~2:10
- **핵심**: T8 (탈출 속도) + T14 (메인 스트레이트 진입) = 랩타임 80%

## 트러블슈팅

### 시트 이동 키가 미러 조정으로 작동
→ Ctrl/Shift stuck. 가상 키보드 (Win + Ctrl + O) 확인

→ 차량 탑승 상태에서 Options → Controls 열기

→ Ctrl + Alt + 키패드 화살표로 가운데 모니터 안으로 이동 후 SimHub Screen Capture
